# Mars Habitat Design Platform

An interactive single-screen STEM classroom platform for sixth-grade students to design a Mars habitat with a fixed total area of 1000 km².

## How to Use

Open `index.html` directly in a browser.

Students can:

- Review population needs
- Set their own map scale
- Draw and edit functional zones
- Place and edit facility icons
- Manually enter actual areas and ratios
- Use presentation mode for classroom sharing

The platform does not automatically calculate area, ratios, or scale conversions. Students complete the mathematical reasoning themselves.

## GitHub Pages

To publish with GitHub Pages:

1. Upload all files in this folder to a GitHub repository.
2. Make sure `index.html` is in the repository root.
3. In GitHub, open `Settings` → `Pages`.
4. Choose the `main` branch and `/root`.
5. Save and open the generated Pages URL.
